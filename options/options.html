    How ScamWarning Works

    1
    Local Analysis

    ScamWarning runs pattern matching on the page content entirely within your browser. No data leaves your device at this step.

    2
    API Lookup (if needed)

    If local patterns detect suspicious signals, the page domain and extracted signals are sent to our assessment API. Full page content is sent only when no cached assessment exists.

    3
    Assessment Result

    The result is cached locally for 1 hour. Most revisits use the local cache with zero network calls.

    What We Collect

    • Domain name, page title, and meta description (when API is called)
    • Extracted threat signal counts (not raw page text, in most cases)
    • Full page text only when local analysis detects threats and no cached result exists

    What We Never Collect

    • Browsing history or page visit timestamps
    • Personal information, names, emails, or accounts
    • Form data, passwords, or credentials
    • Cookies, tokens, or session data
    • IP addresses (not logged by our servers)
